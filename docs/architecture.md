# Kiến Trúc Dự Án Landing Hub

## Tổng Quan Kiến Trúc
- **Frontend**: ReactJS (web), React Native (mobile) với component chung.
- **Backend**: Node.js + Express, kết nối MongoDB.
- **API**: RESTful, hỗ trợ OAuth (Google).

## Thành Phần Chính
- **Web**: Trang chủ, marketplace, AI create, dashboard.
- **Mobile**: Tương tự web, tối ưu cho di động.
- **Backend**: Auth, marketplace, AI pipeline, deploy, leads, email, reports.
